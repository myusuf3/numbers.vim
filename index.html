numbers.vim

better line numbers for vim


Do all kinds of things to it here.

A free, open source This plugin will alternate between relative numbering (normal mode) and absolute numbering (insert mode) depending on the mode you are in. Also provides convenient mappable NumberToggle function.

Requirements

  • Vim 7.3+
  • pathogen

Mappings

nnoremap <F3> :NumbersToggle<CR>


Installation

  • git clone https://github.com/myusuf3/numbers.vim.git ~/.vim/bundle/numbers
